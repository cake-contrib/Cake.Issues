The [Cake.Issues.DocFx addin] provides the following features.

# Basic features

* Reads warnings from [DocFx] log files.

  :::{.alert .alert-info}
  [DocFx] can be run with [Cake.DocFx] addin.
  :::

# Supported comment formats

|                                                                    | Comment format                 | Remarks                        |

The [Cake.Issues.Reporting.Sarif addin]{target="_blank"} provides the following features:

- [x] Creates SARIF compatible files.

Supports the following properties in the SARIF report:

- [x] RuleId
- [x] Message
- [x] Kind
- [x] Level
- [x] Location
- [x] RuleUrl
